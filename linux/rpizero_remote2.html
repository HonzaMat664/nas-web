1. Připojení IR LED na GPIO 27

Ujisti se, že je zapojení správné:

  • Anoda IR LED → odpor (např. 100–220 ohmů) → GPIO 27

  • Katoda → GND

2. Instalace potřebných balíčků

sudo apt update
sudo apt install ir-ctl lirc

4. Odeslání IR příkazu (např. standby)
Pokud znáš kód jako NEC:0x7A1F, použij tento příkaz:
ir-ctl -d /dev/lirc0 -S nec:0x7A1F

Pokud chceš používat zároveň vysílání i příjem:
Musíš aktivovat dva různé piny: jeden pro TX a druhý pro RX.
Do /boot/config.txt přidej:
Například, pokud:
  • LED (vysílání) je na GPIO 27
  • přijímač (např. IR senzor) je na GPIO 17
Přidej:
dtoverlay=gpio-ir,gpio_pin=17
dtoverlay=gpio-ir-tx,gpio_pin=27
oba dtoverlay příkazy spolu kolidují
Návrh spolehlivého řešení: pevně přiřadit názvy přes udev
Můžeme vytvořit pravidlo, které jedno zařízení pojmenuje /dev/lirc-tx a druhé /dev/lirc-rx.
1. Zjisti identifikaci obou zařízení:
Spusť:
udevadm info /dev/lirc0
udevadm info /dev/lirc1

Všímej si řádku s OF_NAME (např. gpio-ir nebo gpio-ir-tx) nebo KERNELS.
1. Vytvoř udev pravidlo
Spusť:

sudo nano /etc/udev/rules.d/99-lirc-names.rules  #  začátek udev

Vlož následující:
udev

SUBSYSTEM=="lirc", DEVPATH=="/devices/platform/gpio-ir-transmitter@1b/rc/rc1/lirc0", SYMLINK+="lirc-tx"
SUBSYSTEM=="lirc", DEVPATH=="/devices/platform/ir-receiver@11/rc/rc0/lirc1", SYMLINK+="lirc-rx"

Ulož soubor (Ctrl+O, Enter, Ctrl+X

2. Načti nová pravidla

sudo udevadm trigger

  • Dlouhá nožka (anoda) LED → odpor (např. 330 Ω) → GPIO24 (pin 18)
  • Krátká nožka (katoda) → GND
2. Instalace potřebných knihoven
Na RPi Zero nainstaluj:
sudo apt update
sudo apt install python3-gpiozero python3-requests -y

3. Python skript pro řízení LED
Ulož si třeba jako volumio_led.py:

#!/usr/bin/env python3
from gpiozero import LED
from time import sleep
import requests

# Nastavení
VOLUMIO_IP = "192.168.1.152"
LED_GPIO = 24
CHECK_INTERVAL = 1  # sekundy

led = LED(LED_GPIO)

def get_state():
    try:
        url = f"http://{VOLUMIO_IP}/api/v1/getState"
        r = requests.get(url, timeout=3)
        return r.json().get("status", "stop")
    except:
        return "stop"

while True:
    state = get_state()

    if state == "play":
        led.on()
    elif state == "pause":
        led.blink(on_time=0.5, off_time=0.5)
    else:
        led.off()

    sleep(CHECK_INTERVAL)

5. Stav ověříš:

Chceš-li je kdykoli ručně restartovat:
sudo systemctl restart indikace
sudo systemctl restart dual

Chceš je zastavit:
sudo systemctl stop indikace
sudo systemctl stop dual

Jak zrušit automatické spouštění (disable) služeb:
sudo systemctl disable indikace
sudo systemctl disable dual

To znamená, že po restartu už se tyto služby automaticky nespustí.
Jak služby zároveň zastavit hned teď:

sudo systemctl stop indikace
sudo systemctl stop dual

Kontrola, že jsou služby zakázané:

systemctl is-enabled indikace
systemctl is-enabled dual
