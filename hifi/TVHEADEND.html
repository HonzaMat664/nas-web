TVHEADEND:

sudo systemctl restart tvheadend
sudo systemctl status tvheadend

připojení USB FLASH:
sudo mkdir -p /mnt/usb # vytvoření adresáře
sudo umount /mnt/usb 2>/dev/null # odpojení disku
sudo mount -o uid=1000,gid=1000,umask=000 /dev/sda1 /mnt/usb # připojení disku, práva pro všechny
ls -ld /mnt/usb # ověření, musíš vidět: drwxrwxrwx  ...  pi  pi  /mnt/usb

spouštění při startu:
sudo nano /etc/fstab   # uprav, přidej řádek nakonec
UUID=B890-A59B /mnt/usb exfat defaults,uid=1000,gid=1000,umask=000 0 0
