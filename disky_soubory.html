Disky a soubory

du (disk usage, využití disku)
du -h --max-depth=1 /home | sort -h # sortování rpi
du -h --max-depth=1 /volumeUSB1/usbshare/music/vol # výpis synology
-s → souhrn složek
-h → human-readable (KB/MB/GB)
sort -hr → seřadí od největší po nejmenší
head -10 → ukáže prvních 10 položek


ncdu (NCurses Disk Usage), ncurses (new curses), CURSor handling → curses.
ncdu /home # spuštění
ncdu -o report.json /home/pi # report


# připojení NAS k LINUX
mkdir -p ~/synology_usb # vytvoření adresáře
sudo mount -t cifs -o username=synology,password=asdf2025jkl,vers=3.0 //192.168.1.149/usbshare1 ~/synology_usb # připojení NAS
ncdu ~/synology_usb # spuštění NCDU
sudo umount ~/usbshare1 # odpojení
