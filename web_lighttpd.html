Instaluj lighttpd

sudo apt update
sudo apt install lighttpd
Vytvoř úvodní HTML stránku
Soubor webové stránky je ve výchozím umístění:
sudo nano /var/www/html/index.html

<!DOCTYPE html>
<html lang="cs">
<head>
  <meta charset="UTF-8">

    body { font-family: sans-serif; text-align: center; margin-top: 100px; }

  </style>

</html>
